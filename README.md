# ZL-MultiMedia Project

This project is built with:
- **Vue 3** (SFC with `<script setup>`)
- **JavaScript** (ESM)
- **Element Plus** (UI Component Library)
- **Vite** (Optimized build tool)
- **Node.js**: v18.20.5 (Ensured compatibility)

## Getting Started

### Prerequisites
Make sure you are using Node.js `18.20.5`.

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Features
- **Premium Design System**: Custom tokens in `src/style.css`
- **Glassmorphism**: UI cards with backdrop-blur and borders
- **Responsiveness**: Mobile-first grid layouts
- **Element Plus Icons**: Fully registered and ready to use
